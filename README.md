# h5-anbosport-hub

## 项目简介

`h5-anbosport-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面彼此独立，不关联任何特定域名或网站，仅作为静态资源存放与版本管理使用。

## 目录说明

```
h5-anbosport-hub/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面所需的公共资源（如样式、脚本、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/` 目录下每个 HTML 文件代表一个独立页面，文件名通常反映其功能或用途。
- `assets/` 目录用于存放页面间的共享资源，避免重复引用。

## 页面归档说明

本仓库中的 HTML 页面以静态方式存放，可直接通过浏览器打开预览，或部署至任意静态托管服务。  
每个页面均为自包含文件（或引用仓库内资源），不依赖外部 API 或数据库。

归档目的包括但不限于：
- 保存页面快照与历史版本
- 便于团队或协作者审阅、测试
- 作为独立功能模块的展示与备份

## 维护说明

- 新增页面请放入 `pages/` 目录，并保持命名清晰、无重复。
- 如有公共资源（CSS、JS、图片等），请优先存放至 `assets/` 目录并统一引用。
- 请勿在页面中嵌入具体域名、营销链接或诱导点击内容。
- 建议每次提交时附带简要的 commit 信息，以便追溯页面变更。

## 使用方式

1. 克隆本仓库：
   ```bash
   git clone https://github.com/your-username/h5-anbosport-hub.git
   ```
2. 打开 `pages/` 下任意 HTML 文件，在浏览器中预览。
3. 如需部署，可将整个仓库内容上传至静态服务器或 GitHub Pages。
